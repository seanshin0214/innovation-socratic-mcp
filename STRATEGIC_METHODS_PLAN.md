# 추가할 전략적 의사결정 & 경영전략 방법론

## 1. 의사결정 도구 (Decision-Making Tools)

### DECISION TREE
- **카테고리**: Strategic
- **단계**: 5단계
- **용도**: 복잡한 의사결정의 각 선택지와 결과 분석
- **질문 구조**:
  1. 결정해야 할 핵심 질문은 무엇입니까?
  2. 가능한 선택지(옵션)는 무엇입니까? (최소 2개 이상)
  3. 각 선택지의 예상 결과는 무엇입니까?
  4. 각 결과의 확률과 가치는? (예: 성공확률 70%, 가치 높음)
  5. 최적의 선택은 무엇이며 그 이유는?

### DECISION MATRIX (Weighted Scoring)
- **카테고리**: Strategic
- **단계**: 4단계
- **용도**: 여러 기준으로 옵션 평가 및 점수화
- **질문 구조**:
  1. 평가할 옵션들은 무엇입니까?
  2. 중요한 평가 기준은 무엇입니까? (예: 비용, 시간, 효과)
  3. 각 기준의 중요도는? (1-10점)
  4. 각 옵션을 각 기준으로 평가하면? (매트릭스 작성)

### COST-BENEFIT ANALYSIS
- **카테고리**: Strategic
- **단계**: 4단계
- **용도**: 투자 대비 효과 분석
- **질문 구조**:
  1. 고려 중인 결정/투자는 무엇입니까?
  2. 예상되는 모든 비용은? (직접비용, 기회비용, 시간)
  3. 예상되는 모든 이익은? (단기, 중기, 장기)
  4. 비용 대비 이익이 정당화됩니까?

### PROS-CONS-FIXES
- **카테고리**: Strategic
- **단계**: 3단계
- **용도**: 장단점 분석 + 단점 해결방안
- **질문 구조**:
  1. PROS: 이 선택의 장점은 무엇입니까?
  2. CONS: 이 선택의 단점/위험은 무엇입니까?
  3. FIXES: 단점을 완화할 방법은 무엇입니까?

## 2. 경영전략 프레임워크 (Business Strategy Frameworks)

### BCG MATRIX (Growth-Share Matrix)
- **카테고리**: Strategic
- **단계**: 4단계
- **용도**: 포트폴리오 분석 (제품/사업부)
- **질문 구조**:
  1. 분석할 제품/사업부는 무엇입니까?
  2. 각각의 시장 성장률은? (높음/낮음)
  3. 각각의 시장 점유율은? (높음/낮음)
  4. 분류 결과 (Star/Cash Cow/Question Mark/Dog)와 전략은?

### SWOT ANALYSIS
- **카테고리**: Strategic
- **단계**: 5단계
- **용도**: 내부 역량 및 외부 환경 분석
- **질문 구조**:
  1. STRENGTHS: 우리의 강점은 무엇입니까?
  2. WEAKNESSES: 우리의 약점은 무엇입니까?
  3. OPPORTUNITIES: 외부 기회는 무엇입니까?
  4. THREATS: 외부 위협은 무엇입니까?
  5. SO/ST/WO/WT 전략은 무엇입니까?

### PORTER'S FIVE FORCES
- **카테고리**: Strategic
- **단계**: 6단계
- **용도**: 산업 경쟁 구조 분석
- **질문 구조**:
  1. 분석할 산업/시장은 무엇입니까?
  2. 기존 경쟁자 간 경쟁 강도는?
  3. 신규 진입자의 위협은?
  4. 대체재의 위협은?
  5. 공급자의 교섭력은?
  6. 구매자의 교섭력은?

### PESTEL ANALYSIS
- **카테고리**: Strategic
- **단계**: 6단계
- **용도**: 거시환경 분석
- **질문 구조**:
  1. POLITICAL: 정치적 요인은?
  2. ECONOMIC: 경제적 요인은?
  3. SOCIAL: 사회문화적 요인은?
  4. TECHNOLOGICAL: 기술적 요인은?
  5. ENVIRONMENTAL: 환경적 요인은?
  6. LEGAL: 법적 요인은?

### ANSOFF MATRIX (Growth Strategy)
- **카테고리**: Strategic
- **단계**: 4단계
- **용도**: 성장전략 선택
- **질문 구조**:
  1. 현재 제품/시장은 무엇입니까?
  2. MARKET PENETRATION: 기존 시장에서 점유율 확대 방법은?
  3. MARKET DEVELOPMENT: 새로운 시장 진출 방법은?
  4. PRODUCT DEVELOPMENT: 신제품 개발 방법은?
  5. DIVERSIFICATION: 다각화 방법은?

### VALUE CHAIN ANALYSIS
- **카테고리**: Strategic
- **단계**: 5단계
- **용도**: 가치사슬 분석으로 경쟁우위 발견
- **질문 구조**:
  1. 주요 활동(Primary Activities)은? (생산, 마케팅, 물류 등)
  2. 지원 활동(Support Activities)은? (인사, 기술, 조달 등)
  3. 각 활동에서 창출되는 가치는?
  4. 비용 대비 가치가 높은 활동은?
  5. 개선이 필요한 활동은?

### BLUE OCEAN STRATEGY
- **카테고리**: Strategic
- **단계**: 4단계
- **용도**: 경쟁 없는 새로운 시장 창출
- **질문 구조**:
  1. ELIMINATE: 업계 표준 중 제거할 요소는?
  2. REDUCE: 업계 표준보다 줄일 요소는?
  3. RAISE: 업계 표준보다 높일 요소는?
  4. CREATE: 업계에 없는 새로 만들 요소는?

### OKR (Objectives and Key Results)
- **카테고리**: Strategic
- **단계**: 3단계
- **용도**: 목표 설정 및 측정
- **질문 구조**:
  1. OBJECTIVE: 달성하고자 하는 명확한 목표는?
  2. KEY RESULTS: 목표 달성을 측정할 핵심 결과는? (3-5개)
  3. INITIATIVES: 핵심 결과를 달성할 구체적 실행방안은?
