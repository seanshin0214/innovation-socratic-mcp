# PESTLE Analysis (PESTLE 분석)

## 개요
조직이나 프로젝트에 영향을 미치는 거시환경 요인을 Political, Economic, Social, Technological, Legal, Environmental 6가지 측면에서 체계적으로 분석하는 전략 도구입니다.

## 명칭 변형
- **PEST**: 기본형 (Political, Economic, Social, Technological)
- **PESTLE/PESTEL**: Legal, Environmental 추가
- **STEEPLE**: Ethical 추가
- **DESTEP**: Demographic 포함

## 6가지 요인 상세

### P - Political (정치적 요인)
| 요인 | 검토 사항 |
|------|----------|
| 정부 정책 | 산업 정책, 규제 방향 |
| 정치 안정성 | 정권 변화, 정치 리스크 |
| 무역 정책 | 관세, FTA, 수출입 규제 |
| 세금 정책 | 법인세, 인센티브 |
| 국제 관계 | 외교, 지정학적 이슈 |

### E - Economic (경제적 요인)
| 요인 | 검토 사항 |
|------|----------|
| 경제 성장률 | GDP, 산업 성장률 |
| 인플레이션 | 물가 상승률, 구매력 |
| 금리 | 대출 비용, 투자 환경 |
| 환율 | 수출입 영향, 해외 투자 |
| 실업률 | 노동 시장 상황 |

### S - Social (사회적 요인)
| 요인 | 검토 사항 |
|------|----------|
| 인구 통계 | 연령, 성별, 인구 변화 |
| 라이프스타일 | 트렌드, 소비 패턴 |
| 문화/가치관 | 사회적 인식, 윤리 |
| 교육 수준 | 인력 품질, 기술 수준 |
| 건강/웰빙 | 건강 의식, 고령화 |

### T - Technological (기술적 요인)
| 요인 | 검토 사항 |
|------|----------|
| 기술 발전 | AI, 자동화, 디지털화 |
| R&D 투자 | 혁신 역량, 기술 경쟁 |
| 기술 인프라 | 인터넷, 5G, 클라우드 |
| 기술 수명 | 기술 진부화 속도 |
| 지적재산권 | 특허, 기술 보호 |

### L - Legal (법률적 요인)
| 요인 | 검토 사항 |
|------|----------|
| 노동법 | 고용, 해고, 근로 조건 |
| 소비자 보호법 | 소비자 권리, 제품 책임 |
| 데이터 보호법 | GDPR, 개인정보 |
| 지적재산권법 | 특허, 저작권 |
| 산업 규제 | 허가, 인증, 표준 |

### E - Environmental (환경적 요인)
| 요인 | 검토 사항 |
|------|----------|
| 기후변화 | 탄소중립, 그린뉴딜 |
| 환경 규제 | 배출 규제, 폐기물 |
| 자원 가용성 | 원자재, 에너지 |
| 지속가능성 | ESG, 친환경 트렌드 |
| 자연재해 | 리스크, 공급망 영향 |

## 혁신이론 적용

### 혁신 환경 스캔 PESTLE

| 요인 | 혁신 관점 질문 |
|------|---------------|
| Political | "정부 정책이 혁신에 어떤 기회/위협?" |
| Economic | "경제 상황이 혁신 투자에 미치는 영향?" |
| Social | "사회 트렌드가 요구하는 혁신은?" |
| Technology | "신기술이 가져오는 혁신 기회는?" |
| Legal | "규제 변화가 혁신에 미치는 영향?" |
| Environment | "환경 이슈가 요구하는 혁신은?" |

```

        ↓

```

## 분석 프로세스

### Step 1: 범위 정의
- 분석 대상 (조직, 프로젝트, 산업)
- 시간 범위 (현재, 3년, 5년)
- 지역 범위 (국내, 글로벌)

### Step 2: 데이터 수집
- 정부 보고서, 통계
- 산업 분석 보고서
- 뉴스, 트렌드 리포트
- 전문가 인터뷰

### Step 3: 요인 분석
- 각 PESTLE 요인별 현황 파악
- 기회와 위협 식별
- 중요도 평가

### Step 4: 시나리오 도출
- 주요 불확실성 식별
- 가능한 시나리오 구성
- 대응 전략 수립

### Step 5: 모니터링
- 핵심 지표 설정
- 정기적 업데이트
- 변화 대응

## 활용 상황
