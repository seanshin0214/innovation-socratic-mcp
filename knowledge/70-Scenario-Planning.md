## 1. 도구 개요

### 정의
시나리오 플래닝은 미래의 불확실성에 대비하기 위해 복수의 그럴듯한 미래 시나리오를 구성하고, 각 시나리오에서의 전략적 대응을 준비하는 방법론

### 핵심 원리
- **다중 미래**: 단일 예측이 아닌 복수의 미래 가능성 탐색
- **불확실성 수용**: 예측 불가능한 미래를 인정
- **전략적 유연성**: 어떤 미래에도 대응 가능한 전략 개발
- **조기 경보**: 각 시나리오의 신호(Signal) 모니터링

### 언제 사용하나?
- 장기 전략 수립
- 불확실성이 높은 환경
- 투자 의사결정
- 위기 대응 계획
- 신사업 기회 탐색
- 정책 수립

## 2. 시나리오 매트릭스

### 2x2 시나리오 매트릭스
가장 일반적인 형태

```
                    핵심 불확실성 1
                 낮음          높음
핵심          ┌─────────┬─────────┐
불확실성    높│ 시나리오 │ 시나리오 │
   2       음│    A    │    B    │
             ├─────────┼─────────┤
           낮│ 시나리오 │ 시나리오 │
           음│    C    │    D    │
             └─────────┴─────────┘
```

### 시나리오 예시
**핵심 불확실성**:
1. 기술 발전 속도 (빠름/느림)
2. 규제 환경 (강화/완화)

```
              규제 강화         규제 완화
기술    ┌─────────────┬─────────────┐
빠름    │ "통제된 혁신"│ "자유로운   │
        │             │  혁명"      │
        ├─────────────┼─────────────┤
기술    │ "정체된     │ "점진적     │
느림    │  보수"      │  개방"      │
        └─────────────┴─────────────┘
```

## 3. 진행 단계

### Phase 1: 초점 질문 정의 (30분)
- 어떤 결정을 위한 시나리오인가?
- 시간 범위는? (3년, 10년, 30년)
- 지리적 범위는?

### Phase 2: 핵심 동인 파악 (60분)
외부 환경 요인 분석 (STEEP):
- **S**ocial: 사회/인구통계 변화
- **T**echnological: 기술 발전
- **E**conomic: 경제 동향
- **E**nvironmental: 환경 변화
- **P**olitical: 정치/규제 변화

### Phase 3: 핵심 불확실성 선정 (45분)
동인 중 가장 불확실하고 영향력 큰 2개 선택:
```
영향력    높음 │ ★ ★ ★
              │   ↑선택
         낮음 │ ○ ○ ○
              └──────────
               낮음   높음
                불확실성
```

### Phase 4: 시나리오 구성 (90분)
각 시나리오별로:
- 이름 부여 (기억하기 쉽게)
- 스토리라인 작성
- 주요 특징 정의
- 2030년 이 시나리오의 세상은?

### Phase 5: 시나리오별 전략 개발 (60분)
- 각 시나리오에서 우리 조직의 상황은?
- 필요한 역량/자원은?
- 기회와 위협은?
- 대응 전략은?

### Phase 6: 모니터링 지표 설정 (30분)
- 어떤 시나리오로 가고 있는지 알려주는 신호
- 조기 경보 지표 (Early Warning Signals)

## 6. 전략 로버스트니스 테스트

각 전략을 모든 시나리오에서 평가:

| 전략 | 시나리오A | 시나리오B | 시나리오C | 시나리오D | 로버스트 점수 |
|------|-----------|-----------|-----------|-----------|--------------|
| 전략1 | O | O | X | O | 3/4 |
| 전략2 | O | X | O | O | 3/4 |
| 전략3 | O | O | O | O | 4/4 (강건) |

**로버스트 전략**: 어떤 시나리오에서도 유효한 전략

## 7. 실전 예시: AI 시대 대학 전략

### 핵심 불확실성
1. AI 대체율 (높음/낮음)
2. 대면 교육 수요 (증가/감소)

### 4가지 시나리오

**시나리오 A: "AI 캠퍼스"**
- AI 대체율 높음 + 대면 수요 증가
- AI가 교육하지만 캠퍼스 경험 중시
- 전략: 체험형 캠퍼스 + AI 튜터

**시나리오 B: "완전 온라인"**
- AI 대체율 높음 + 대면 수요 감소
- AI 기반 완전 원격 교육
- 전략: 플랫폼 사업 전환

**시나리오 C: "전통 회귀"**
- AI 대체율 낮음 + 대면 수요 증가
- 인간 교수 중심 대면 교육 부활
- 전략: 소규모 세미나 강화

**시나리오 D: "현상 유지"**
- AI 대체율 낮음 + 대면 수요 감소
- 현재 온/오프라인 혼합 지속
- 전략: 하이브리드 최적화

## 8. 주의사항

### Do's
- 극단적 시나리오도 포함
- 각 시나리오를 동등하게 취급
- 정기적으로 시나리오 업데이트
- 조기 경보 신호 지속 모니터링

### Don'ts
- 선호하는 시나리오에 편향
- 가장 가능성 높은 시나리오만 집중
- 시나리오를 예측으로 오해
- 한 번 만들고 방치
