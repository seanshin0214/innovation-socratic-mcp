# SWOT Analysis (SWOT 분석)

## 개요
조직이나 프로젝트의 강점(Strengths), 약점(Weaknesses), 기회(Opportunities), 위협(Threats)을 체계적으로 분석하는 전략 기획 도구입니다.

## 기원
- **Stanford Research Institute** (1960-70년대)
- Albert Humphrey가 Fortune 500 기업 연구 중 개발
- 전략 경영의 기본 도구로 자리잡음

## SWOT 매트릭스

```
              긍정적 요인          부정적 요인
           ┌─────────────────┬─────────────────┐
           │                 │                 │
   내부    │   Strengths     │   Weaknesses    │
   요인    │     (강점)       │     (약점)       │
           │                 │                 │
           ├─────────────────┼─────────────────┤
           │                 │                 │
   외부    │  Opportunities  │    Threats      │
   요인    │     (기회)       │     (위협)       │
           │                 │                 │
           └─────────────────┴─────────────────┘
```

## 각 요소 상세

### S - Strengths (강점)
**"우리가 잘하는 것은 무엇인가?"**
- 경쟁 우위 요소
- 핵심 역량
- 차별화 포인트
- 보유 자원/자산

**검토 질문:**
- 우리만의 독특한 자원은?
- 경쟁사보다 뛰어난 점은?
- 고객이 우리를 선택하는 이유는?
- 핵심 역량과 노하우는?

### W - Weaknesses (약점)
**"개선이 필요한 것은 무엇인가?"**
- 경쟁 열위 요소
- 자원 부족
- 역량 격차
- 개선 필요 영역

**검토 질문:**
- 피해야 할 것은?
- 경쟁사보다 못한 점은?
- 고객이 불만족하는 점은?
- 부족한 자원이나 역량은?

### O - Opportunities (기회)
**"활용할 수 있는 외부 환경은?"**
- 시장 성장
- 기술 변화
- 규제 완화
- 사회/문화 트렌드

**검토 질문:**
- 시장의 새로운 트렌드는?
- 기술 발전이 가져오는 기회는?
- 경쟁사의 실수나 공백은?
- 정부 정책 변화의 이점은?

### T - Threats (위협)
**"주의해야 할 외부 위험은?"**
- 경쟁 심화
- 시장 축소
- 규제 강화
- 기술 대체

**검토 질문:**
- 경쟁자들의 움직임은?
- 불리한 시장 변화는?
- 기술 변화가 주는 위협은?
- 규제/법률 변화의 영향은?

## TOWS 전략 매트릭스

SWOT 분석 후 전략 도출:

```
                    Strengths (S)        Weaknesses (W)
                  ┌─────────────────┬─────────────────┐
                  │                 │                 │
   Opportunities  │   SO 전략        │   WO 전략        │
        (O)       │ (강점으로 기회   │ (약점 보완하여   │
                  │  극대화)         │  기회 포착)      │
                  ├─────────────────┼─────────────────┤
                  │                 │                 │
      Threats     │   ST 전략        │   WT 전략        │
        (T)       │ (강점으로 위협   │ (약점 최소화,    │
                  │  회피/극복)      │  위협 회피)      │
                  └─────────────────┴─────────────────┘
```

| 전략 | 설명 | 예시 |
|------|------|------|
| SO (공격) | 강점으로 기회 극대화 | 기술력으로 신시장 진출 |
| WO (보완) | 약점 보완하여 기회 포착 | 파트너십으로 역량 보완 |
| ST (방어) | 강점으로 위협 대응 | 브랜드력으로 경쟁 방어 |
| WT (철수) | 약점 최소화, 위협 회피 | 비핵심 사업 철수 |

## 혁신이론 적용 예시

## 워크샵 진행 방법

### 준비물
- 화이트보드 또는 플립차트
- 포스트잇 4색
- 마커펜

### 진행 순서
1. **개별 작업** (10분): 각자 4가지 요소 브레인스토밍
2. **팀 공유** (20분): 포스트잇으로 벽에 부착, 그룹핑
3. **우선순위화** (15분): 투표로 핵심 요소 선정
4. **TOWS 전략** (30분): 교차 분석으로 전략 도출
5. **액션 플랜** (15분): 구체적 실행 계획

## 장점과 한계

### 장점
- 간단하고 직관적
- 다양한 상황에 적용 가능
- 팀 토론 촉진
- 전체적 시각 제공

### 한계
- 정성적 분석에 치우침
- 요소 간 상호작용 무시
- 우선순위 결정 어려움
- 현재 상태에 집중 (동적 변화 반영 어려움)
