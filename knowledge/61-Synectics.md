# Synectics (시넥틱스)

## 개요
William J.J. Gordon이 개발한 창의적 문제해결 기법으로, 유추(Analogy)와 은유(Metaphor)를 사용하여 익숙한 것을 낯설게, 낯선 것을 익숙하게 만들어 새로운 관점을 얻습니다.

## 창시자
- **William J.J. Gordon** (1961)
- 저서: "Synectics: The Development of Creative Capacity"
- Arthur D. Little 컨설팅 그룹에서 개발
- 그리스어 synektikos (다른 요소들을 결합하다)에서 유래

```
│   낯설게 만들기    │   익숙하게 만들기    │

│ 보기             │ 연결하기            │
└───────────────────┴─────────────────────┘
```

## 4가지 유추 유형

### 1. Direct Analogy (직접 유추)
**"자연이나 다른 분야에서 비슷한 것은?"**

실제 세계의 유사한 시스템에서 영감

**예시:**
- 벨크로 ← 도꼬마리 씨앗
- 총알열차 ← 물총새 부리
- 에어백 ← 복어가 부풀어 오르는 것

**질문:**
- "자연에서 이 문제를 어떻게 해결하는가?"
- "다른 산업에서 비슷한 문제를 어떻게 해결했는가?"

### 2. Personal Analogy (개인 유추)
**"내가 그것이 된다면?"**

문제의 요소가 되어 그 관점에서 경험

**예시:**
- "내가 스프링이라면 어떤 느낌일까?"
- "내가 고객의 불만이라면?"
- "내가 데이터라면 어디로 흘러가고 싶을까?"

**질문:**
- "만약 내가 ____라면, 어떻게 느낄까?"
- "그것의 관점에서 세상은 어떻게 보일까?"

### 3. Symbolic Analogy (상징 유추)
**"이것을 상징적으로 표현한다면?"**

압축된 이미지나 시적 표현으로 본질 포착

**예시:**
- 혁신 = "창조적 파괴"
- 스타트업 = "정글의 새싹"
- 리더십 = "오케스트라 지휘자"

**질문:**
- "이 문제를 시적으로 표현한다면?"
- "2단어로 본질을 표현한다면?"
- "어떤 역설이 있는가?"

### 4. Fantasy Analogy (환상 유추)
**"마법처럼 해결된다면?"**

현실 제약 없이 이상적 해결책 상상

**예시:**
- "기계가 스스로 수리한다면?"
- "고객이 원하는 것을 읽을 수 있다면?"
- "제품이 사용자에게 말을 건다면?"

**질문:**
- "마법 지팡이가 있다면?"
- "완벽한 해결책은 어떤 모습일까?"
- "물리 법칙을 무시할 수 있다면?"

## 혁신이론 적용

### 자연에서 배우는 혁신

| 자연 현상 | 혁신 적용 |
|----------|----------|
| 거미줄의 강도 | 초경량 고강도 소재 |
| 상어 피부 | 항균 표면 |
| 개미 군집 지능 | 물류 최적화 알고리즘 |
| 나뭇잎 광합성 | 태양광 에너지 |
| 게코 발바닥 | 접착 기술 |

### 혁신 문제에 Synectics 적용

| 유추 유형 | 혁신 질문 |
|----------|----------|
| Direct | "다른 산업에서 이 문제를 어떻게 해결했는가?" |
| Personal | "내가 고객이라면 이 혁신을 어떻게 느낄까?" |
| Symbolic | "이 혁신을 2단어로 표현한다면?" |
| Fantasy | "제약 없이 이상적 혁신은 어떤 모습인가?" |

## Synectics 세션 진행

### Step 1: 문제 정의 (PAU)
- Problem As Understood
- 문제를 명확히 진술

### Step 2: 문제 재진술 (PAG)
- Problem As Given → Problem As Goals
- "어떻게 하면 ___할 수 있을까?"

### Step 3: 유추 탐색
- 4가지 유추 유형 적용
- 먼 영역에서 영감 찾기

### Step 4: 강제 연결
- 유추에서 얻은 아이디어를
- 원래 문제에 강제로 연결

### Step 5: 해결책 발전
- 실현 가능한 형태로 발전
- 구체화 및 평가

## 활용 예시

### 예시: 회의 효율성 개선

**문제**: "회의가 비효율적이다"
**HMW**: "어떻게 하면 회의를 더 효율적으로 만들 수 있을까?"

| 유추 유형 | 유추 | 적용 |
|----------|------|------|
| Direct | 벌집의 의사소통 (춤) | 핵심 정보만 전달 |
| Personal | "내가 시간이라면" | 시간의 소중함 인식 |
| Symbolic | "아이디어 수확" | 결과물 중심 |
| Fantasy | "생각 읽기" | 사전 공유 문서 |
