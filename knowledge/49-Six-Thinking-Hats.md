# Six Thinking Hats (여섯 색깔 모자 기법)

## 개요
Edward de Bono가 개발한 병렬적 사고 기법으로, 6가지 색깔의 모자가 각각 다른 사고 방식을 상징합니다. 한 번에 하나의 관점에서 집중적으로 사고하여 효과적인 의사결정과 문제해결을 도모합니다.

## 창시자
- **Edward de Bono** (1985)
- 저서: "Six Thinking Hats"

## 6가지 모자별 역할

### 🎩 White Hat (흰색 모자) - 사실/데이터
**"우리가 알고 있는 것은 무엇인가?"**
- 객관적 사실과 수치
- 확인된 정보와 데이터
- 정보의 공백 파악
- 감정이나 해석 배제

**혁신이론 적용 질문:**
- "이 이론의 실증 연구 결과는?"
- "시장 데이터가 무엇을 말하는가?"

### 🎩 Red Hat (빨간 모자) - 감정/직관
**"이것에 대해 어떻게 느끼는가?"**
- 감정, 직관, 본능
- 논리적 정당화 불필요
- 즉각적 반응
- 미적 판단

**혁신이론 적용 질문:**
- "이 혁신에 대한 조직원들의 감정은?"
- "고객이 첫인상으로 어떻게 느낄까?"

### 🎩 Black Hat (검은 모자) - 비판/리스크
**"왜 이것이 작동하지 않을 수 있는가?"**
- 리스크와 위험 요소
- 잠재적 문제점
- 논리적 비판
- 주의 사항

**혁신이론 적용 질문:**
- "이 전략의 실패 가능성은?"
- "법적, 윤리적 리스크는?"

### 🎩 Yellow Hat (노란 모자) - 긍정/기회
**"왜 이것이 작동할 수 있는가?"**
- 이점과 가치
- 긍정적 측면
- 기회 발견
- 낙관적 전망

**혁신이론 적용 질문:**
- "성공 시 얻는 이점은?"
- "숨겨진 기회는 무엇인가?"

### 🎩 Green Hat (초록 모자) - 창의/대안
**"다른 방법은 없는가?"**
- 창의적 아이디어
- 대안 탐색
- 새로운 가능성
- 제약 없는 발상

**혁신이론 적용 질문:**
- "다른 적용 방법은 없나?"
- "완전히 새로운 접근법은?"

### 🎩 Blue Hat (파란 모자) - 프로세스/통제
**"우리가 어디에 있고 다음은 무엇인가?"**
- 사고 프로세스 관리
- 회의 진행
- 요약 및 결론
- 다음 단계 설정

**혁신이론 적용 질문:**
- "실행 단계를 어떻게 설계할까?"
- "의사결정 프로세스는?"

## 활용 순서 예시

### 기본 순서
```
Blue → White → Green → Yellow → Black → Red → Blue
(정의) (사실) (아이디어) (장점) (단점) (감정) (결론)
```

### 아이디어 평가 순서
```
Blue → White → Yellow → Black → Green → Red → Blue
(정의) (사실) (장점) (단점) (대안) (감정) (결론)
```

### 문제 해결 순서
```
Blue → White → Black → Green → Yellow → Red → Blue
(정의) (사실) (문제) (해결) (기회) (감정) (결론)
```

## 워크샵 진행 방법

### 개인 사용
1. 문제/주제 정의 (Blue)
2. 순차적으로 각 모자 관점에서 사고
3. 노트에 각 모자별 아이디어 기록
4. 종합 및 결론 도출 (Blue)

### 그룹 사용
1. 퍼실리테이터가 Blue Hat 역할
2. 전체 그룹이 동시에 같은 모자 착용
3. 시간 제한 (모자당 3-5분)
4. 모든 의견 시각화 (화이트보드/포스트잇)
