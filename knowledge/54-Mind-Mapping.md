# Mind Mapping (마인드맵)

## 개요
Tony Buzan이 체계화한 방사형 사고 도구로, 중심 아이디어에서 가지를 뻗어나가며 연관 개념들을 시각적으로 연결합니다. 두뇌의 자연스러운 사고 방식을 반영합니다.

## 창시자
- **Tony Buzan** (1974)
- 저서: "The Mind Map Book"
- 전 세계 3억 명 이상 사용

## 기본 구조

```
                          ┌── 세부 3-1
                     ┌─ 가지 3 ─┤
                     │         └── 세부 3-2
              ┌── 가지 2
              │      └── 세부 2-1
         ┌────┤
         │    └── 가지 1 ─── 세부 1-1
    ┌────┴────┐
    │ 중심    │
    │ 주제    │
    └────┬────┘
         │    ┌── 가지 4 ─── 세부 4-1
         │    │
         └────┼── 가지 5 ─┬── 세부 5-1
              │          └── 세부 5-2
              └── 가지 6
```

## 핵심 원칙 (Buzan's Rules)

### 1. 중심 이미지
- 페이지 중앙에 배치
- 3가지 이상 색상 사용
- 이미지나 아이콘 활용

### 2. 가지 (Branches)
- 중심에서 방사형으로 뻗어나감
- 굵은 선 → 가는 선 (위계)
- 유기적 곡선 (직선 X)
- 각 가지에 다른 색상

### 3. 키워드
- 가지 위에 단일 키워드
- 문장이 아닌 핵심 단어
- 인쇄체로 명확하게

### 4. 이미지
- 가능한 많은 이미지/아이콘
- 기억력 향상 효과
- 창의성 자극

### 5. 연결
- 관련 가지 간 연결선
- 관계 표시
- 통합적 사고 촉진

## 마인드맵 작성 단계

### Step 1: 준비
- A4 또는 A3 종이 (가로 방향)
- 색연필/마커 (최소 4색)
- 또는 디지털 도구

### Step 2: 중심 주제
- 페이지 중앙에 주제 작성
- 이미지로 표현 권장
- 원이나 테두리로 강조

### Step 3: 주요 가지
- 중심에서 방사형으로 굵은 가지
- 각 가지에 핵심 카테고리
- 5-7개 주요 가지 권장

### Step 4: 세부 가지
- 주요 가지에서 세부 가지 확장
- 점점 가는 선
- 키워드와 이미지

### Step 5: 연결 및 완성
- 관련 개념 간 연결선
- 색상으로 그룹핑
- 전체 검토 및 보완

## 혁신이론 적용

### 이론 연결망 시각화
```
                    ┌── 파괴적 혁신
               ┌─ Christensen ─┤
               │              └── 혁신가의 딜레마
         ┌─────┤
         │     └─ Schumpeter ─── 창조적 파괴
    ┌────┴────┐
    │ 혁신    │
    │ 이론    │
    └────┬────┘
         │     ┌─ Rogers ─── 혁신확산이론
         │     │
         └─────┼─ Chesbrough ─── 개방형 혁신
               │
               └─ Ries ─── 린 스타트업
```

### 아이디어 확장용 질문
| 가지 방향 | 질문 |
|----------|------|
| 원인/배경 | "왜 이것이 필요한가?" |
| 결과/영향 | "이것이 가져올 결과는?" |
| 구성요소 | "이것은 무엇으로 구성되는가?" |
| 유사/연관 | "이것과 비슷한 것은?" |
| 반대/대안 | "이것의 반대는?" |
| 적용/활용 | "이것을 어디에 적용할 수 있나?" |

## 활용 유형

### 1. 브레인스토밍 마인드맵
- 자유로운 아이디어 발산
- 양 중심, 판단 보류
- 연상 작용 활용

### 2. 정리/요약 마인드맵
- 책, 강의, 회의 내용 정리
- 핵심 구조 파악
- 기억 보조

### 3. 계획 마인드맵
- 프로젝트 계획
- To-do 리스트
- 의사결정 분석

### 4. 발표/프레젠테이션 마인드맵
- 발표 흐름 구조화
- 청중 이해 촉진
- 암기 부담 감소

- 새로운 연결 발견
- 재미있고 참여적
