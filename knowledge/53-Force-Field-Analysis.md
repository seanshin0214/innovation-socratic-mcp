# Force Field Analysis (힘의 장 분석)

## 개요
Kurt Lewin이 개발한 변화 관리 도구로, 변화를 촉진하는 힘(Driving Forces)과 변화에 저항하는 힘(Restraining Forces)을 시각화하여 변화 전략을 수립합니다.

## 창시자
- **Kurt Lewin** (1943)
- 사회심리학의 아버지
- 변화의 3단계 모델 (Unfreeze-Change-Refreeze) 개발자

## 핵심 개념

### Lewin의 변화 공식
```
C = D × V × F > R

C = Change (변화)
D = Dissatisfaction (현재에 대한 불만)
V = Vision (미래 비전)
F = First steps (첫 실행 단계)
R = Resistance (저항)
```

**변화가 일어나려면: 추진력 > 저항력**

## 힘의 장 다이어그램

```
    추진력 (Driving Forces)          저항력 (Restraining Forces)
    
    시장 경쟁 압력     ════►  ◄════  변화에 대한 두려움
                         ║
    기술 발전        ═══════►  ◄═══════  기존 시스템 투자
                         ║
    고객 요구 변화   ════════►  ◄════════  조직 관성
                         ║
    비용 절감 필요    ═════►  ◄═════  역량 부족
                         ║
    리더십 지원     ════════►  ◄════════  정치적 저항
                         │
                    ┌────┴────┐
                    │ 현재 상태 │
                    │(균형점)  │
                    └─────────┘
```

## 혁신/조직 변화 적용

### 추진력 (Driving Forces) 예시
| 범주 | 추진력 |
|------|--------|
| 외부 압력 | 경쟁 심화, 규제 변화, 시장 트렌드 |
| 내부 동기 | 비전, 리더십, 성장 욕구 |
| 자원 | 예산 확보, 기술 도입, 인재 영입 |
| 성과 | 위기의식, 성공 사례, 보상 체계 |

### 저항력 (Restraining Forces) 예시
| 범주 | 저항력 |
|------|--------|
| 심리적 | 두려움, 불확실성, 습관 |
| 조직적 | 관료주의, 사일로, 정치 |
| 자원적 | 시간 부족, 예산 제약, 역량 부족 |
| 기술적 | 레거시 시스템, 호환성 문제 |

## 분석 단계

### Step 1: 목표 정의
- 원하는 변화 상태 명확히 정의
- 현재 상태와의 갭 파악

### Step 2: 힘 식별
- 추진력 브레인스토밍
- 저항력 브레인스토밍
- 각각 5-10개 식별

### Step 3: 힘의 강도 평가
- 1-5점 척도로 각 힘의 강도 평가
- 화살표 길이로 시각화

### Step 4: 전략 수립
- **방법 1**: 추진력 강화
- **방법 2**: 저항력 약화 (더 효과적!)
- **방법 3**: 새로운 추진력 추가

### Step 5: 액션 플랜
- 각 힘에 대한 구체적 조치
- 책임자 및 일정 지정

## 혁신이론 적용 질문

| 질문 유형 | Force Field 적용 |
|----------|------------------|
| 변화 동인 | "혁신을 추진하는 힘은 무엇인가?" |
| 장애물 | "혁신을 방해하는 힘은 무엇인가?" |
| 전략 | "저항을 줄이려면 어떻게 해야 하는가?" |
| 우선순위 | "가장 강력한 저항력은 무엇인가?" |

## Lewin의 3단계 변화 모델과 연계

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   Unfreeze  │ →  │   Change    │ →  │  Refreeze   │
│   (해빙)     │    │   (변화)     │    │   (재동결)   │
├─────────────┤    ├─────────────┤    ├─────────────┤
│ - 위기의식   │    │ - 새로운     │    │ - 새 행동    │
│   조성      │    │   행동 도입   │    │   강화      │
│ - 저항력    │    │ - 추진력     │    │ - 제도화     │
│   약화      │    │   활용      │    │             │
└─────────────┘    └─────────────┘    └─────────────┘
```

## 워크샵 진행 팁

### 준비물
- 대형 화이트보드
- 포스트잇 (2색: 추진력/저항력)
- 점수 스티커

### 진행 순서
1. 변화 목표 공유 (5분)
2. 개별 브레인스토밍 (10분)
3. 팀 공유 및 그룹핑 (15분)
4. 강도 평가 투표 (10분)
5. 전략 토론 (20분)
6. 액션 플랜 (10분)
