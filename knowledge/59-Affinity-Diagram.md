# Affinity Diagram (친화도법 / KJ법)

## 개요
Jiro Kawakita가 개발한 아이디어 정리 기법으로, 대량의 정성적 데이터나 아이디어를 자연스러운 친화성에 따라 그룹핑하여 패턴과 통찰을 발견합니다.

## 창시자
- **Kawakita Jiro** (川喜田二郎, 1967)
- 일본 문화인류학자
- 현장 조사 데이터 정리를 위해 개발
- "KJ법"이라고도 불림

## 핵심 원리

```
혼란스러운 아이디어들          친화도로 그룹핑           패턴 발견
┌─────────────────┐        ┌─────────────────┐      ┌─────────────────┐
│ ○ ● □ △ ◇      │        │ [그룹 A]         │      │ ┌───────────┐   │
│   ○  □  ●       │   →    │  ○ ○ ○          │  →   │ │ 인사이트 1│   │
│ △  ◇  ○  ●     │        │ [그룹 B]         │      │ └───────────┘   │
│   □  △  ◇      │        │  □ □            │      │ ┌───────────┐   │
└─────────────────┘        │ [그룹 C]         │      │ │ 인사이트 2│   │
                           │  △ △ ◇          │      │ └───────────┘   │
                           └─────────────────┘      └─────────────────┘
```

## 진행 단계

### Step 1: 데이터 수집
- 브레인스토밍, 인터뷰, 관찰 등에서 수집
- 각 아이디어를 포스트잇에 하나씩 작성
- 20-100개 아이디어 권장

### Step 2: 무작위 배치
- 벽이나 테이블에 모든 포스트잇 배치
- 순서나 패턴 없이 무작위로

### Step 3: 침묵의 그룹핑
- 말없이 관련 있어 보이는 것들 모으기
- 직관에 따라 자연스럽게
- 정답 없음, 느낌으로 분류

### Step 4: 그룹 이동 허용
- 다른 사람이 옮긴 것 다시 옮겨도 OK
- 합의가 될 때까지 반복
- 단독 아이디어도 OK (하나의 그룹)

### Step 5: 그룹 명명
- 각 그룹에 헤더 카드 작성
- 그룹의 본질을 나타내는 이름
- 동사형 또는 문장 권장

### Step 6: 관계 파악
- 그룹 간 관계 표시
- 화살표, 선으로 연결
- 상위 그룹으로 통합 가능

### Step 7: 통찰 도출
- 패턴에서 인사이트 발견
- 액션 아이템 도출
- 문서화

## 혁신이론 적용

### 혁신 아이디어 분류 예시

```
[고객 경험 혁신]          [프로세스 혁신]         [기술 혁신]
├─ 앱 UI 개선            ├─ 자동화 도입          ├─ AI 적용
├─ 고객 응대 개선         ├─ 결재 단계 축소       ├─ 클라우드 전환
├─ 셀프서비스            ├─ 병목 제거           ├─ API 통합
└─ 맞춤 추천             └─ 표준화              └─ 보안 강화
```

### 혁신 장애 요인 분석

| 그룹 | 아이디어 예시 |
|------|-------------|
| 조직 문화 | 위험 회피, 단기 성과 압박, 사일로 |
| 자원 부족 | 예산 제약, 인력 부족, 시간 부족 |
| 역량 격차 | 기술 역량, 변화 관리, 리더십 |
| 외부 환경 | 규제, 시장 불확실성, 경쟁 |

## 워크샵 진행 팁

### 준비물
- 포스트잇 (2-3색)
- 마커펜 (굵은 것)
- 큰 벽면 또는 테이블
- 테이프

### 시간 배분
1. 아이디어 작성 (10-15분)
2. 침묵의 그룹핑 (15-20분)
3. 그룹 명명 (10분)
4. 관계 파악 (10분)
5. 토론 및 인사이트 (15분)

### 진행 규칙
- ✅ 침묵 유지 (그룹핑 단계)
- ✅ 모든 아이디어 존중
- ✅ 직관적으로 분류
- ✅ 완벽할 필요 없음
- ❌ 논쟁 금지
- ❌ 강제 분류 금지
