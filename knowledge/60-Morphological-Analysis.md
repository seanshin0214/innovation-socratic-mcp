# Morphological Analysis (형태학적 분석)

## 개요
Fritz Zwicky가 개발한 체계적 혁신 기법으로, 문제의 모든 차원과 각 차원의 가능한 옵션들을 매트릭스로 나열하여 새로운 조합을 발견합니다.

## 창시자
- **Fritz Zwicky** (1940년대)
- 스위스 천문학자/항공우주 과학자
- 제트 추진 연구소에서 개발
- 우주탐사 문제 해결에 적용

## 핵심 원리

```
문제를 차원으로 분해 → 각 차원의 옵션 나열 → 새로운 조합 발견

차원1: [옵션A] [옵션B] [옵션C]
차원2: [옵션A] [옵션B] [옵션C]
차원3: [옵션A] [옵션B] [옵션C]
       ↓        ↓       ↓
       조합1    조합2    조합3...
```

## Morphological Box (형태 상자)

### 기본 구조

```
┌────────────┬─────────┬─────────┬─────────┬─────────┐
│  차원/속성  │ 옵션 1   │ 옵션 2   │ 옵션 3   │ 옵션 4   │
├────────────┼─────────┼─────────┼─────────┼─────────┤
│  재료       │ 플라스틱 │ 금속    │ 목재    │ 복합재   │
├────────────┼─────────┼─────────┼─────────┼─────────┤
│  크기       │ 소형    │ 중형    │ 대형    │ 맞춤형   │
├────────────┼─────────┼─────────┼─────────┼─────────┤
│  동력       │ 수동    │ 전기    │ 배터리  │ 태양광   │
├────────────┼─────────┼─────────┼─────────┼─────────┤
│  연결방식   │ 유선    │ 무선    │ 블루투스 │ WiFi    │
└────────────┴─────────┴─────────┴─────────┴─────────┘

가능한 조합 수 = 4 × 4 × 4 × 4 = 256가지
```

## 진행 단계

### Step 1: 문제/제품 정의
- 분석할 문제나 제품 명확히 정의
- 예: "새로운 휴대용 조명 기기"

### Step 2: 차원(Parameter) 식별
- 문제의 핵심 차원/속성 나열
- 5-10개 차원 권장
- 예: 형태, 재료, 동력원, 제어방식, 용도

### Step 3: 각 차원의 옵션 나열
- 각 차원에서 가능한 모든 옵션
- 3-5개 옵션씩 권장
- 기존 옵션 + 새로운 가능성

### Step 4: Morphological Box 작성
- 매트릭스 형태로 정리
- 행: 차원, 열: 옵션

### Step 5: 조합 탐색
- 각 차원에서 하나씩 선택하여 조합
- 무작위 또는 체계적 탐색
- 흥미로운 조합 표시

### Step 6: 평가 및 선정
- 실현 가능성 평가
- 혁신성 평가
- 유망한 조합 선정

## 혁신이론 적용

### 혁신 전략 Morphological Box

| 차원 | 옵션 1 | 옵션 2 | 옵션 3 | 옵션 4 |
|------|--------|--------|--------|--------|
| 혁신 유형 | 제품 | 프로세스 | 비즈니스 모델 | 서비스 |
| 혁신 수준 | 점진적 | 급진적 | 파괴적 | 아키텍처 |
| 대상 시장 | 기존 고객 | 신규 고객 | 비고객 | 글로벌 |
| 기술 접근 | 내부 R&D | 인수 | 파트너십 | 오픈소스 |
| 실행 속도 | 빠름 | 중간 | 느림 | 단계적 |

### 조합 예시
`[제품] × [파괴적] × [비고객] × [파트너십] × [빠름]`
→ "비고객 대상 파괴적 제품 혁신을 파트너십으로 빠르게 실행"

## 실전 예시

### 예시: 새로운 음료 제품 개발

| 차원 | 옵션 1 | 옵션 2 | 옵션 3 | 옵션 4 |
|------|--------|--------|--------|--------|
| 베이스 | 물 | 과일즙 | 우유 | 차 |
| 기능 | 에너지 | 수분보충 | 영양보충 | 해독 |
| 포장 | 캔 | 페트병 | 파우치 | 분말 |
| 타겟 | 청소년 | 직장인 | 운동선수 | 시니어 |
| 가격대 | 저가 | 중가 | 고가 | 프리미엄 |

**새로운 조합**:
`[차] × [에너지] × [분말] × [직장인] × [프리미엄]`
→ "직장인용 프리미엄 에너지 분말 차"

## 조합 수 계산

```
총 조합 수 = 옵션1 수 × 옵션2 수 × ... × 옵션n 수

예: 5개 차원, 각 4개 옵션 = 4^5 = 1,024 조합
```

## 조합 탐색 전략

### 1. 무작위 탐색
- 주사위나 난수 생성기 사용
- 예상치 못한 조합 발견

### 2. 체계적 탐색
- 모든 조합 순차 검토
- 완전성 보장 (소규모만 가능)

### 3. 휴리스틱 탐색
- 제약 조건으로 불가능 조합 제외
- 유망한 영역 집중

### 4. 크로스 일관성 분석 (CCA)
- 옵션 간 양립 가능성 검토
- 불가능 조합 자동 제외
