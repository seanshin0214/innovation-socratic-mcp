# PMI (Plus/Minus/Interesting)

## 개요
Edward de Bono가 개발한 균형 잡힌 평가 도구로, 아이디어나 결정에 대해 긍정적(Plus), 부정적(Minus), 흥미로운(Interesting) 측면을 체계적으로 검토합니다.

## 창시자
- **Edward de Bono**
- CoRT 사고력 프로그램의 일부
- Six Thinking Hats 저자

## PMI 프레임워크

```
┌─────────────────────────────────────────┐
│                  PMI                     │
├─────────────┬─────────────┬─────────────┤
│     P       │      M      │      I      │
│   Plus      │    Minus    │ Interesting │
│   (장점)     │    (단점)    │   (흥미)     │
├─────────────┼─────────────┼─────────────┤
│ 긍정적인    │ 부정적인     │ 중립적이지만 │
│ 측면       │ 측면        │ 주목할 점    │
│            │             │             │
│ 왜 좋은가?  │ 왜 나쁜가?   │ 흥미로운    │
│            │             │ 가능성은?    │
└─────────────┴─────────────┴─────────────┘
```

## 각 요소 상세

### P - Plus (장점)
**"이것의 좋은 점은 무엇인가?"**
- 이점, 혜택
- 강점, 가치
- 성공 가능성
- 긍정적 결과

### M - Minus (단점)
**"이것의 나쁜 점은 무엇인가?"**
- 문제점, 위험
- 비용, 손실
- 장애물, 제약
- 부정적 결과

### I - Interesting (흥미로운 점)
**"흥미롭거나 주목할 만한 점은?"**
- 가능성, 기회
- 질문, 의문
- 탐구할 가치
- 예상치 못한 결과
- 판단 보류 영역

## 실전 예시

### 예시: "회사에 재택근무 도입"

| Plus (장점) | Minus (단점) | Interesting (흥미) |
|------------|-------------|-------------------|
| 통근 시간 절약 | 협업 어려움 | 생산성 변화는? |
| 일-생활 균형 | 소통 비효율 | 조직문화 변화? |
| 사무실 비용 절감 | 보안 위험 | 새로운 도구 필요? |
| 인재 풀 확대 | 관리 어려움 | 하이브리드 모델은? |
| 집중력 향상 | 고립감 | 장기적 영향은? |

## 혁신이론 적용

### 혁신안 평가용 질문

| 영역 | 질문 |
|------|------|
| Plus | "이 혁신의 핵심 가치와 이점은?" |
| Minus | "실행 시 예상되는 리스크와 비용은?" |
| Interesting | "탐구해볼 만한 새로운 가능성은?" |

### 혁신 의사결정 PMI

```
혁신안: _________________________________

P (+)                M (-)               I (?)

└─────────────┘    └─────────────┘    └─────────────┘
```

## 진행 방법

### 개인 사용
1. 평가 대상 명확히 정의
2. P, M, I 각각 2-3분씩 집중 (순서대로)
3. 각 항목 최소 3개 이상 작성
4. 전체 검토 및 결론 도출

### 그룹 사용
1. 주제 공유 (2분)
2. 개별 PMI 작성 (5분)
3. 팀 공유 및 통합 (10분)
4. 토론 및 결론 (10분)

## PMI vs 다른 도구

| 도구 | 초점 | 사용 시점 |
|------|------|----------|
| **PMI** | 균형 잡힌 평가 | 빠른 아이디어 검토 |
| **SWOT** | 전략적 분석 | 전략 수립 |
| **Six Hats** | 다중 관점 | 심층 의사결정 |
| **Pros/Cons** | 이분법 | 단순 비교 |

## Interesting (I)의 가치

PMI의 핵심은 "Interesting" 영역:
- 단순한 장단점 비교를 넘어섬
- 판단 보류 영역 인정
- 새로운 탐구 방향 제시
- 창의적 사고 자극
- 예상치 못한 기회 발견

## 활용 상황
- 아이디어 초기 평가
- 의사결정 전 검토
- 회의 토론 구조화
- 프로젝트 제안서 검토
- 개인 결정 (진로, 투자 등)

## 장점
- 간단하고 빠름 (5-10분)
- 균형 잡힌 시각
- I 영역으로 창의성 자극
- 감정적 판단 방지
- 누구나 쉽게 사용
