# Brainwriting 6-3-5

## 개요
Bernd Rohrbach가 개발한 구조화된 아이디어 생성 기법으로, 6명이 3개의 아이디어를 5분마다 작성하고 돌려가며 발전시킵니다. 30분 만에 108개 아이디어를 생성할 수 있습니다.

## 창시자
- **Bernd Rohrbach** (1969)
- 독일 마케팅 컨설턴트
- "침묵의 브레인스토밍"

## 6-3-5 의미

```
┌─────────────────────────────────────┐
│              6 - 3 - 5              │
│                                     │
│   6명의 참가자                       │
│   3개의 아이디어 (매 라운드)          │
│   5분 시간 제한                      │
│                                     │
│   = 6 × 3 × 6라운드 = 108개 아이디어  │
└─────────────────────────────────────┘
```

## 진행 방법

### 준비물
- 6-3-5 워크시트 (6장)
- 펜
- 타이머
- 조용한 환경

### Step 1: 문제 정의 (5분)
- 해결할 문제 명확히 정의
- 모든 참가자가 이해

### Step 2: 첫 번째 라운드 (5분)
- 각자 워크시트 첫 번째 줄에 3개 아이디어 작성
- 말하지 않고 조용히 작성

### Step 3: 시트 전달
- 5분 후 오른쪽 사람에게 시트 전달
- 이전 아이디어 읽기

### Step 4: 반복 (5분 × 5라운드)
- 이전 아이디어를 보고 영감 받아 새 아이디어 작성
- 발전, 변형, 결합, 새로운 아이디어 모두 OK
- 6라운드까지 반복

### Step 5: 정리 및 평가 (15분)
- 모든 아이디어 공유
- 그룹핑 및 우선순위화
- 실행 가능한 아이디어 선정

## 6-3-5 워크시트

```
문제: _________________________________

참가자: _____________   날짜: ___________

┌─────────┬─────────────┬─────────────┬─────────────┐
│ 라운드   │  아이디어 1  │  아이디어 2  │  아이디어 3  │
├─────────┼─────────────┼─────────────┼─────────────┤
│ 1 (5분) │             │             │             │
├─────────┼─────────────┼─────────────┼─────────────┤
│ 2 (5분) │             │             │             │
├─────────┼─────────────┼─────────────┼─────────────┤
│ 3 (5분) │             │             │             │
├─────────┼─────────────┼─────────────┼─────────────┤
│ 4 (5분) │             │             │             │
├─────────┼─────────────┼─────────────┼─────────────┤
│ 5 (5분) │             │             │             │
├─────────┼─────────────┼─────────────┼─────────────┤
│ 6 (5분) │             │             │             │
└─────────┴─────────────┴─────────────┴─────────────┘
```

## 혁신이론 적용

### 혁신 아이디어 발굴용 질문

| 라운드 | 초점 | 예시 |
|--------|------|------|
| 1 | 현재 상태 개선 | 기존 프로세스 개선 아이디어 |
| 2 | 기술 적용 | AI/자동화 적용 방안 |
| 3 | 고객 관점 | 고객 경험 혁신 |
| 4 | 비용 효율 | 비용 절감 방안 |
| 5 | 새로운 기회 | 신규 시장/서비스 |
| 6 | 통합/확장 | 아이디어 결합 |

## 변형 버전

### 4-4-4 (소규모)
- 4명, 4개 아이디어, 4분
- 64개 아이디어 생성

### 10-3-3 (대규모)
- 10명, 3개 아이디어, 3분
- 90개 아이디어 생성

### 온라인 버전
- 디지털 화이트보드 사용
- Miro, MURAL, FigJam
- 비동기적 진행 가능

## 장점 vs 일반 브레인스토밍

| 측면 | Brainwriting 6-3-5 | 일반 브레인스토밍 |
|------|-------------------|-----------------|
| 참여도 | 전원 동등 참여 | 외향적 성향 우세 |
| 영향력 | 상사/권위 영향 최소화 | 지위 영향 받음 |
| 아이디어 수 | 108개 (구조적) | 불확실 |
| 시간 | 30분 고정 | 가변적 |
| 발전 | 아이디어 위에 쌓기 | 개별 아이디어 |
| 분위기 | 조용하고 집중 | 시끄럽고 활발 |

## 진행 팁

### DO
- ✅ 시간 엄격히 관리
- ✅ 조용한 환경 유지
- ✅ 이전 아이디어 읽고 영감 받기
- ✅ 양 우선 (판단 보류)
- ✅ 글씨 알아보기 쉽게

### DON'T
- ❌ 말로 토론하기
- ❌ 아이디어 비판
- ❌ 너무 자세히 쓰기
- ❌ 이전 아이디어 무시
- ❌ 시간 초과 허용
